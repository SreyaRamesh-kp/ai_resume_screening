# AI Resume Screening System

## Overview
The AI Resume Screening System is a machine learning application that evaluates resumes by comparing them with a job description. It uses a Hugging Face Sentence Transformer model to calculate semantic similarity and rank resumes based on their relevance.

## Features
- Upload resumes in PDF format
- Extract text from resumes
- Compare resumes with a job description
- AI-based semantic similarity using Hugging Face
- Resume ranking based on matching score
- Simple and user-friendly interface

## Technologies Used
- Python
- Hugging Face Sentence Transformers
- PyMuPDF (fitz)
- Scikit-learn
- NumPy
- Flask
- HTML & CSS

## Model
**Hugging Face Model:**
- `sentence-transformers/all-MiniLM-L6-v2`



## How It Works
1. Enter the job description.
2. Upload one or more resumes in PDF format.
3. The system extracts text from each resume.
4. AI computes semantic similarity between the resume and the job description.
5. Resumes are ranked based on their matching score.
